Deep Natural Language Processing

Level: Master
Language: English
Degree: Data Science and Engineering
Period: 2021-ongoing
Semester: Winter (September/January)
More info: https://github.com/MorenoLaQuatra/DeepNLP

Teaching assistant both for in-class and lab practices. An unextensive list of course's topics is reported below:

  • Fundamentals of text processing
  • Word- and Sentence-level vector spaces
  • Deep learning architectures (ELMo, BERT and beyond)
  • Overview and practices with NLP tasks: Machine Translation, Sentiment Analysis, Text Summarization, IR, Chatbots
The final exam also include a final project:
  • Student projects 2021-2022


Fundamentals of Databases

Level: Bachelor
Language: Italian
Degree: Computer and Management Engineering
Period: 2019-ongoing
Semester: Winter (September/January)
More info: Course website


This course is an introduction to databases for undergraduate students. The training activities include the following topics:
  • Relational Algebra
  • SQL language
  • No-SQL concepts (Mongo DB)
  • Dashboard generation with Google Data Studio
